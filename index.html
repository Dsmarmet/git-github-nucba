Git Github Basico

git init para inicializar el repositorio git
git status para ver que no esta agregado
ls -h para ver carpetas ocultas
cd para volver atras
git log para ver los cambios
git add . para agregar los no agregados
git commit -m "-nombre-" para confirmar cambios

Github en la terminal

git remote add origin -url del repositorio- para subir al repositorio
git branch -M -nombre de la rama-
git push -u origin -nombre de la rama- para crear la rama o guardar cambios

Crear otra rama en hithub

git checkout -b -el nombre de la rama que quieras-
git checkout -nombre de la rama- para ver la rama

Copiar codigo de una rama a otra

git merge (ubicarce en la rama a la que le querramos pegar el codigo de la otra) -nombre de la rama que vamos a copiar-
git diff para ver diferencias en los codigos
git branch -d -nombre de la rama- para borrar rama
